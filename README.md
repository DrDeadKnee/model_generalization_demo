# model_generalization_demo
Short tutorial on getting models to generalize


### Installation
First clcone the repo:
```bash
git clone https://github.com/DrDeadKnee/model_generalization_demo.git
```
Then install virtualenv if you haven't already:
```bash
python3 -m pip install virtualenv
```

Finally, create a virtual environment with the required libraries and activate:
```bash
python3 -m venv env
source env/bin/activate
pip install requirements.txt
```
